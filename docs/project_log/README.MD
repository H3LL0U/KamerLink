# Project log

This folder consists of the following files

## WORK_LOG.MD

Contains the following for different dates

- The time when the work was being done
- How much time was spent
- What was done in that period of time

## DECISIONS.MD

Contains the decisions made when constructing the project.

That includes subjects such as:

- The project architecture
- The project stack (i.e database, frontend, backend)
- The main project idea
- Problems that occurred during development
- Sources
